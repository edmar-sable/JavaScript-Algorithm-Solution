Algorithm 2 Determining Triangles Given the lengths of three sticks A, B, and C of different lengths, A triangle can be formed with these three sticks. If yes, output “YES”, otherwise output “NO”. ▣ Input description The first line gives the lengths of less than 100 different A, B, and C bars. ▣ Output Description Print “YES” or “NO” on the first line. ▣ Input Example 1 6 7 11 ▣ Output example 1 YES ▣ Input Example 2 13 33 17 ▣ Output example 2 NO
